# 3. Sea ORM

Date: 2025-11-03

## Status

Accepted

## Context

[Sea ORM](https://www.sea-ql.org/SeaORM/) provides a nice full async support
while at the same time enables simultaneous support for multiple database
backends (PostgreSQL, MySQL, ...).

## Decision

Sea ORM is used as the ORM library for the database communication.

## Consequences

No known
